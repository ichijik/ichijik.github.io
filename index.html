バーチャルカントリー計画

参加型オンライン国家創造プロジェクトに関する公式情報

計画概要

「参加者ひとりひとりが国を創る。形のないオンラインコミュニティ。」
これが、『バーチャルカントリー計画』の核となる理念です。

本計画は、ルールも目的も設定されていないゼロ地点から、参加者の対話と行動を通じて、新たな社会が自律的に形成される過程を観測するものです。皆さんの想像力とコミュニケーションが、そのままこの共同体の歴史と文化となります。

GitHub プロフィール

本計画は、誰も見たことのない「共同創造」の景色を観測したいという、純粋な好奇心から発足しました。私は王や神ではなく、皆さんと同じく、この共同体の一住人であり、その自律的な発展を期待する者です。
